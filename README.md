# 📊 Pharmacovigilance Safety Monitoring Dashboard

## 🔍 Project Overview
This project presents an interactive Power BI dashboard designed to monitor adverse drug reactions (ADRs) using Individual Case Safety Report (ICSR) data. The dashboard enables identification of high-risk drugs and supports pharmacovigilance decision-making.

---

## 🎯 Objectives
- Analyze adverse event trends over time
- Identify drugs with high case volume and severity
- Evaluate risk using Serious % and Fatality %
- Support early signal detection in drug safety

---

## 🛠️ Tools & Technologies
- Power BI
- DAX
- Excel

---

## 📈 Key Metrics
- Total Cases
- Serious Cases
- Fatal Cases
- Serious %
- Fatality %

---

## 📊 Dashboard Features
- Monthly trend of total vs serious cases
- Drug-wise case analysis
- Serious % by Drug (risk indicator)
- SOC and reaction distribution
- Country-level reporting analysis
- Age group severity breakdown

---

## 🔥 Key Insights
- Certain drugs (e.g., Ibuprofen, Atorvastatin) show high serious case proportions
- General disorders and skin disorders dominate reported SOC categories
- Serious cases show variation across age groups and regions
